# Marketplace Unidades Residenciales

Sistema marketplace para unidades residenciales donde los residentes pueden vender productos, ofrecer servicios y realizar transacciones dentro de su conjunto.

## Clases Principales

- **UnidadResidencial**: Representa el conjunto residencial (edificio, torre, barrio)
- **Marketplace**: Espacio de intercambio asociado a cada unidad
- **Usuario**: Residente que puede vender y comprar
- **Producto**: Artículo físico en venta
- **Servicio**: Servicio ofrecido por residentes
- **Categoría**: Clasificación de productos y servicios
- **Carrito**: Items seleccionados antes de comprar
- **Transacción**: Compra formalizada

## Estructura

```
marketplace/
├── unidad_residencial.py
├── marketplace.py
├── usuario.py
├── producto.py
├── servicio.py
├── categoria.py
├── transaccion.py
└── carrito.py
```

## Uso

```bash
python main.py
```
